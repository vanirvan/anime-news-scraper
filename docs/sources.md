## How to Analyze a Source

### Step 1: Check for RSS

1. Go to the website
2. Look for RSS icon in browser address bar or footer
3. Try common RSS URLs:
   - `/rss`
   - `/rss.xml`
   - `/feed`
   - `/feed.xml`
   - `/atom.xml`
4. View page source and search for:
   ```html
   <link rel="alternate" type="application/rss+xml"
   ```

### Step 2: Test RSS Feed

1. Open RSS URL in browser
2. Verify it returns valid XML
3. Check what content is included:
   - Title?
   - Description (snippet)?
   - Full content?
   - Images?
   - Categories?

### Step 3: Analyze Article Structure

1. Open any article from the site
2. Open Browser DevTools (F12)
3. Use Select Element tool (arrow icon)
4. Click on each element and note the selector:

```
Title:      Right-click > Copy > Copy selector
Content:    Find main article body
Image:      Find main/featured image
Author:     Find author name element
Date:       Find publish date element
Tags:       Find category/tag elements
```

### Step 4: Document Selectors

Fill in the JSON selector object for each source:

```json
{
  "title": "h1.article-title",
  "content": "div.article-body",
  "image": "img.featured-image",
  "author": "span.author-name",
  "date": "time.publish-date",
  "tags": "a.tag"
}
```

### Step 5: Test Selectors

In browser console (F12 > Console):

```javascript
// Test title selector
document.querySelector('h1.article-title')?.textContent

// Test content selector
document.querySelector('div.article-body')?.textContent.substring(0, 100)

// Test image selector
document.querySelector('img.featured-image')?.src
```

## Quick Reference: CSS Selector Types

| Selector | Example | Matches |
|----------|---------|---------|
| Tag | `h1` | All `<h1>` elements |
| Class | `.article-title` | Elements with class="article-title" |
| ID | `#main-content` | Element with id="main-content" |
| Attribute | `[data-type="news"]` | Elements with attribute |
| Descendant | `article p` | `<p>` inside `<article>` |
| Child | `article > p` | Direct child `<p>` of `<article>` |
| Multiple | `h1, h2, h3` | Any of these elements |
